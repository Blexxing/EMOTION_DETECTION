# Emotion Detection App (FER Pretrained Model)

This is a **Streamlit app** that detects emotions in images using a **pretrained FER model**.  

## Features
- Upload an image (jpg, jpeg, png)
- Detect emotions on faces
- Show dominant emotion and probability
